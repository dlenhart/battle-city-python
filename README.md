# battle-city-python
Battle City re-written in Python for game preservation.

This is an exercise in using Claude to convert the C++ game to Python.  The original game is an online multiplayer game where you build a city and ORB 
your enemies for points.  For the sake of this game I'm going to port over the existing game play and feel (doing my best to remember the game from the 90's and 
resorting to some documentation in the c++ repo).  I will likely make this iteration a single player game with in game
AI enemies. I'm going to keep this as close as possible, however I'll likely make my own changes to the control panel and reconfigure the point system
to something that makes more sense for single player.  How I do difficulties with this, will be TBD.
